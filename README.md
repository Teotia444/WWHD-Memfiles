# WWHD-Memfiles and savefiles

Compatible memfiles with the aroma hacker



Memfiles instructions: place the contents of the memfiles folder folder in %appdata%/WWHDHacker/Memfiles (or click "Open Memfile Folder and paste them")

&nbsp;	Remember for memfiles to use REMOVE VOIDOUT if you ever void, otherwise memfiles may break.



Savefiles instructions: place the contents of the savefiles folder in %appdata%/WWHDHacker/Savesfiles (or click "Open Savefiles Dir" and paste them")
