# WWHD-Memfiles and savefiles

Compatible memfiles with the aroma hacker



Memfiles instructions: place the contents of the memfiles folder folder in %appdata%/WWHDHacker/Memfiles (or click "Open Memfile Folder and paste them")

 	Remember for memfiles to use REMOVE VOIDOUT if you ever void, otherwise memfiles may break.



Savefiles instructions: place the contents of each savefiles folder in %appdata%/WWHDHacker/Savesfiles (or click "Open Savefiles Dir" and paste them"). Make sure you paste the individual folders inside "any%" and "any% no mss" folders
